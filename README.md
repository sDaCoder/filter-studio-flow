# Filter Studio Flow

![Cover image](public/cover.png)

A small, fast image filtering and editing web app built with Vite, React, and TypeScript.

Features

- Apply and tweak image filters in real time
- Export and download images
- Preset filters and sliders for customization

Getting Started

1. Install dependencies

```bash
bun install
```

2. Run the dev server

```bash
bun run dev
```

Build

```bash
bun run build
```
